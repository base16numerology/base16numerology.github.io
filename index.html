Once upon a time, the soul spoke in stories, symbols, and dreams. Then we invented computers. Somewhere between myth and machine, your life can be expressed as characters, bytes, and timestamps.

Base 16 Numerology treats modern computing primitives—hex, Unicode bytes, Unix time, and a small open‑source‑style hash—as a symbolic language. It collapses inputs into a single hex digit (0–f), then reads that digit as one of sixteen archetypes.

What it is

This is not a claim that the universe is “proven” to run on hex. It’s a structured self‑reflection system that asks a slightly impolite question:

If the universe really wanted to talk to you, would it use Roman numerals—or hex?

To the devoted numerologist, these digits can feel like the language of the soul. To the geek, they’re modular arithmetic with lore. Both perspectives are welcome.

How it works (computable, on purpose)

The book’s core idea is simple: translate your inputs into the same substrate computers use, then reduce them into one archetypal symbol.

input → bytes (ASCII/Unicode) → LIBRE‑256 hash → mod 16 → one hex digit (0..f)

That one digit can appear in different “slots” (Life Path, Expression, Soul Urge, Personality, Maturity, cycles and event signatures), giving multiple angles on the same underlying pattern.

This site intentionally omits the “app” link for now; the book explains the concepts and interpretation.

Skeptic mode

If you want to approach this like an experiment, here’s the rule: don’t ask “is this objectively true?”—ask “does this describe something useful?”

For believers, it can feel like sacred decoding. For skeptics, it’s a symbolic mirror built from Unicode, hashes, and hex.
